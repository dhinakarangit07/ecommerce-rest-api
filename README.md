# Django Ecommerce API

Full Stack Ecommerce Web Application built using Django REST Framework, MySQL, Bootstrap, and JavaScript.

## Features

* Product Listing
* Product Detail Page
* Add To Cart
* Cart Management
* Order Placement
* Checkout System
* Responsive UI
* Django REST APIs
* MySQL Database Integration

## Technologies Used

* Python
* Django
* Django REST Framework
* MySQL
* Bootstrap 5
* JavaScript
* HTML/CSS
* Docker

## Project Structure

* Products API
* Cart API
* Orders API
* Frontend Templates
* Static Files
* Media Uploads

## Installation

```bash
git clone <repository-url>

cd ecommerce_api

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

## API Endpoints

### Products

* `/api/products/`
* `/api/products/<id>/`

### Cart

* `/api/cart/`

### Orders

* `/api/orders/place/`
